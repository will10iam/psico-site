# 🧠 Site institucional para Psicóloga Emilaine Lopes

Site institucional desenvolvido para uma psicóloga, com o objetivo de apresentar serviços, informações profissionais e facilitar o contato com pacientes.

O projeto foi construído utilizando **Next.js e Tailwind CSS**, focando em performance, design moderno e boa experiência do usuário.

---

## 🚀 Tecnologias utilizadas

- Next.js
- React
- Tailwind CSS
- JavaScript

---

## 🎯 Funcionalidades

- Apresentação profissional
- Sessão de serviços oferecidos
- Informações de contato
- Design responsivo
- Navegação em página única (Single Page)
